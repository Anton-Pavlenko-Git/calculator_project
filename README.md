# Консольный калькулятор

Простая консольная утилита для вычисления базовых арифметических операций (сложение, вычитание, умножение, деление, возведение в степень, факториал) под Linux.
Собирается с помощью CMake
cmake -B build -S .
cmake --build build
Принимает два целых цисла и математические выражения в качестве аргументов командной строки, и выводит результат.
